# Task Archive

This folder contains historical task plans, migration designs, and execution documents that are no longer part of the active task surface.

## Active Sources Of Truth

Use these files before consulting archived task plans:

1. `tasks/production/current-state-audit.md`
2. `tasks/production/overview.md`
3. `tasks/production/progress.md`
4. `memory/auth.md`
5. `memory/memory-security.md`
6. current code and deployment configuration

## What Stays Active Outside Archive

- `tasks/lessons.md`
- active production tracker files in `tasks/production/`
- any future execution docs that reflect current verified work

## Archive Policy

Documents are moved here when they are useful as history or design context but are stale enough to be misleading as active plans.
